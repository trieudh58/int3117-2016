#Nhận xét bài tập Lê Đức Anh


##Bài Tập 1
1. Đạt được
  * Readme khá rõ ràng
  * SourceCode đúng chuẩn, dễ hiểu

2. Chưa đạt được
  * Nên để file readme trong mục BT1
  * Có áp dụng kĩ thuật kiểm thử biên nhưng chưa nêu rõ lí do áp dụng.
  * Bỏ bớt những file không cần thiết

2. Chưa được
